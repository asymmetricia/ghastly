
# Ghastly is the **G**olang **H**ome **AS**sistant **T**oo**L**, **Y**o.

```
ghastly is a tool for interacting with homeassistant

Usage:
  ghastly [command]

Available Commands:
  config      sub-commands for manipulating and interacting with config entries
  device      sub-commands for manipulating and interacting with devices
  entity      sub-commands for manipulating and interacting with entities
  help        Help about any command
  raw         send a raw request and print the result

Flags:
  -h, --help              help for ghastly
      --loglevel string   log level; one of TRACE, DEBUG, INFO, WARN, ERROR, FATAL, PANIC (default "INFO")
      --server string     the URL used to access homeassistant
      --token string      the bearer token used to authenticate to homeassistant. defaults to value of HASS_TOKEN environment variable

Use "ghastly [command] --help" for more information about a command.
```
